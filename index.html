Adding a callback to a button


That button doesn't do anything by default.

We need to create a function, and tell the button to run it when it is clicked.

This div has an id of second-div, and I already added a class red to my custom CSS file.

function toggleRed() {
  $('#second-div').toggleClass('red');
}

Next we will attach that to the button, which has an id of my-button

$('#my-button').on('click', toggleRed)
